Proč Haste?

Proč Haskell?

  • čistě funkcionální jazyk
  • silně typový ← TypeScript
  • expresivní ← CoffeeScript
  • vysokoúrovňový
  • modulární ← pure
  • ⇒ méně chyb
    • Haskell and bugs

Instalace

Nainstalovaná haskell-platform

cabal install haste-compiler
haste-boot

Vývojová verze z githubu

Kompilace

hastec soubor.hs # klientská část
ghc soubor.hs # serverová část
